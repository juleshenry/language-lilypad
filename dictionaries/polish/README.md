# Polish Dictionary Sources

Potential sources for Polish dictionary data:

1.  **Słownik języka polskiego (PWN)**: The official Polish dictionary portal.
    - URL: https://sjp.pwn.pl/
2.  **FreeDict (GitHub)**: Polish-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-pol-eng
3.  **Wiktionary (Polish Section)**: Large collection of Polish lexical entries.
    - URL: https://pl.wiktionary.org/
