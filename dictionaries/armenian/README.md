# Armenian Dictionary Sources

Potential sources for Armenian dictionary data:

1.  **Nayiri.com**: A comprehensive portal for Armenian dictionaries (Eastern and Western).
    - URL: http://www.nayiri.com/
2.  **FreeDict (GitHub)**: Armenian-English and other dictionary pairs in digital format.
    - URL: https://github.com/freedict/fd-arm-eng
3.  **Wiktionary (Armenian Section)**: Multilingual dictionary with extensive Armenian entries.
    - URL: https://en.wiktionary.org/wiki/Category:Armenian_language
