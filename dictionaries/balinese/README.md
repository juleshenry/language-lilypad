# Balinese Dictionary Sources

Potential sources for Balinese (Basa Bali) dictionary data:

1.  **Balinese-English Dictionary (Glosbe)**: Large multilingual dictionary portal.
    - URL: https://glosbe.com/ban/en
2.  **WikiBasa (GitHub)**: Open-source Balinese dictionary and language projects.
    - URL: https://github.com/
3.  **Wiktionary (Balinese Section)**: Community entries for the Balinese language.
    - URL: https://en.wiktionary.org/wiki/Category:Balinese_language
