# Tibetan Dictionary Sources

Potential sources for Tibetan dictionary data:

1.  **Christian Steinert's Tibetan Dictionary**: A massive compilation of Tibetan dictionaries.
    - URL: https://github.com/ctsteinert/tibetan-dictionary-data
2.  **Nitartha Tibetan-English Dictionary**: Reputable academic online dictionary.
    - URL: http://www.nitartha.org/dictionary.html
3.  **Wiktionary (Tibetan Section)**: Collaborative entries for the Tibetan language.
    - URL: https://en.wiktionary.org/wiki/Category:Tibetan_language
