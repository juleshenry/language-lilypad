# Dzongkha Dictionary Sources

Potential sources for Dzongkha dictionary data:

1.  **Dzongkha Development Commission (DDC)**: Official dictionary and language resources for Bhutan.
    - URL: http://www.dzongkha.gov.bt/
2.  **Dzongkha-English Dictionary (GitHub)**: Open-source word lists for Dzongkha.
    - URL: https://github.com/
3.  **Wiktionary (Dzongkha Section)**: Collaborative entries for the Dzongkha language.
    - URL: https://en.wiktionary.org/wiki/Category:Dzongkha_language
