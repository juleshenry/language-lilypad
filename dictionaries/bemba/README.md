# Bemba Dictionary Sources

Potential sources for Bemba (Chibemba) dictionary data:

1.  **Bemba Online Dictionary**: A digital resource for the Bemba language.
    - URL: https://bemba.webonary.org/
2.  **Bemba Word Lists (GitHub)**: Repositories containing Zambian linguistic data.
    - URL: https://github.com/
3.  **Wiktionary (Bemba Section)**: Community entries for the Bemba language.
    - URL: https://en.wiktionary.org/wiki/Category:Bemba_language
