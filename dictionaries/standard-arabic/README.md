# Standard Arabic Dictionary Sources

Potential sources for Modern Standard Arabic (MSA) dictionary data:

1.  **Lughatuna (Living Arabic)**: Comprehensive dictionary for MSA and dialects.
    - URL: https://www.livingarabic.com/
2.  **FreeDict (GitHub)**: Arabic-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-ara-eng
3.  **Wiktionary (Arabic Section)**: Huge collection of Arabic words and morphological data.
    - URL: https://en.wiktionary.org/wiki/Category:Arabic_language
