# Danish Dictionary Sources

Potential sources for Danish dictionary data:

1.  **Den Danske Ordbog (DDO)**: Official modern Danish dictionary.
    - URL: https://ordnet.dk/ddo
2.  **FreeDict (GitHub)**: Danish-English dictionary pairs.
    - URL: https://github.com/freedict/fd-dan-eng
3.  **Wiktionary (Danish Section)**: Comprehensive open-source Danish dictionary.
    - URL: https://en.wiktionary.org/wiki/Category:Danish_language
