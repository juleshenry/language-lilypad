# Tamil Dictionary Sources

Potential sources for Tamil dictionary data:

1.  **Tamil Lexicon (University of Madras)**: The authoritative Tamil dictionary portal.
    - URL: http://dsalsrv02.uchicago.edu/dictionaries/tamil-lex/
2.  **FreeDict (GitHub)**: Tamil-English dictionary data.
    - URL: https://github.com/freedict/fd-tam-eng
3.  **Wiktionary (Tamil Section)**: Extensive open-source Tamil dictionary entries.
    - URL: https://ta.wiktionary.org/
