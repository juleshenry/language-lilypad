# Japanese Dictionary Sources

Potential sources for Japanese dictionary data:

1.  **Jisho.org**: A powerful and user-friendly Japanese-English dictionary.
    - URL: https://jisho.org/
2.  **JMdict (GitHub)**: The core dictionary file used by many Japanese language projects.
    - URL: https://github.com/FooSoft/jmdict
3.  **Wiktionary (Japanese Section)**: Comprehensive collaborative dictionary for Japanese kanji and words.
    - URL: https://en.wiktionary.org/wiki/Category:Japanese_language
