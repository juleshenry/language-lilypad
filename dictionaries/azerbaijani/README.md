# Azerbaijani Dictionary Sources

Potential sources for Azerbaijani dictionary data:

1.  **AzerDict**: A comprehensive online Azerbaijani-English / Azerbaijani-Russian dictionary.
    - URL: https://azerdict.com/
2.  **FreeDict (GitHub)**: Azerbaijani dictionary datasets in digital format.
    - URL: https://github.com/freedict/fd-aze-eng
3.  **Wiktionary (Azerbaijani Section)**: Extensive open-source Azerbaijani dictionary entries.
    - URL: https://en.wiktionary.org/wiki/Category:Azerbaijani_language
