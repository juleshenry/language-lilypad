# Asturian Dictionary Sources

Potential sources for Asturian dictionary data:

1.  **Diccionariu de la Llingua Asturiana (DALLA)**: The official dictionary of the Academy of the Asturian Language.
    - URL: https://academiadelallingua.com/diccionariu/
2.  **Asturian-English Dictionary (GitHub)**: Repositories for Asturian linguistic data.
    - URL: https://github.com/
3.  **Wiktionary (Asturian Section)**: Collaborative dictionary entries for the Asturian language.
    - URL: https://ast.wiktionary.org/
