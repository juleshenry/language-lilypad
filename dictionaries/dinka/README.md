# Dinka Dictionary Sources

Potential sources for Dinka dictionary data:

1.  **Open Dinka Dictionary**: Digital resources for the Dinka language.
    - URL: https://dinka.webonary.org/
2.  **Dinka Language Dataset (GitHub)**: Repositories for South Sudanese language data.
    - URL: https://github.com/
3.  **Wiktionary (Dinka Section)**: Collaborative dictionary entries for Dinka.
    - URL: https://en.wiktionary.org/wiki/Category:Dinka_language
