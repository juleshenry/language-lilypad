# South Levantine Arabic Dictionary Sources

Potential sources for South Levantine Arabic dictionary data:

1.  **Lughatuna (Palestinian/Jordanian)**: Comprehensive dictionary for Levantine dialects.
    - URL: https://www.livingarabic.com/
2.  **Levantine Arabic Project (GitHub)**: Repositories for Arabic dialects from Palestine and Jordan.
    - URL: https://github.com/
3.  **Wiktionary (South Levantine Arabic Section)**: Community entries for the South Levantine Arabic dialect.
    - URL: https://en.wiktionary.org/wiki/Category:South_Levantine_Arabic_language
