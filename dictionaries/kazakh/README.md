# Kazakh Dictionary Sources

Potential sources for Kazakh dictionary data:

1.  **Sozdik.kz**: A popular online Kazakh-Russian-English dictionary.
    - URL: https://sozdik.kz/
2.  **Kazakh Language Projects (GitHub)**: Repositories for Turkic linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Kazakh Section)**: Collaborative dictionary entries for the Kazakh language.
    - URL: https://kk.wiktionary.org/
