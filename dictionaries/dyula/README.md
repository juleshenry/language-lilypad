# Dyula Dictionary Sources

Potential sources for Dyula (Jula) dictionary data:

1.  **Dyula-French Dictionary (Glosbe)**: Large multilingual dictionary dataset.
    - URL: https://glosbe.com/dyu/fr
2.  **Dyula Language Resources (GitHub)**: Repositories containing word lists for Dyula.
    - URL: https://github.com/
3.  **Wiktionary (Dyula Section)**: Community entries for the Dyula language.
    - URL: https://en.wiktionary.org/wiki/Category:Dyula_language
