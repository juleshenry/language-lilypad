# French Dictionary Sources

Potential sources for French dictionary data:

1.  **Le Robert / Larousse**: Official or highly reputable online French dictionaries.
    - URL: https://www.larousse.fr/dictionnaires/francais
2.  **FreeDict (GitHub)**: Large collection of French-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-fra-eng
3.  **Wiktionary (French Section)**: One of the largest and most active Wiktionary sections.
    - URL: https://fr.wiktionary.org/
