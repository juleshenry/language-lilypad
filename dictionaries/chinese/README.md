# Chinese Dictionary Sources

Potential sources for Chinese dictionary data:

1.  **MDBG Chinese-English Dictionary**: A popular online dictionary for Mandarin Chinese.
    - URL: https://www.mdbg.net/chinese/dictionary
2.  **CC-CEDICT (GitHub)**: A community-maintained English to Chinese dictionary project.
    - URL: https://github.com/mdbg/cc-cedict
3.  **Wiktionary (Chinese Section)**: Comprehensive collaborative dictionary for Chinese characters and words.
    - URL: https://en.wiktionary.org/wiki/Category:Chinese_language
