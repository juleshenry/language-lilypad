# Kashmiri Dictionary Sources

Potential sources for Kashmiri dictionary data:

1.  **Kashmiri-English Dictionary (Shabdkosh)**: Popular online dictionary for the Kashmiri language.
    - URL: https://www.shabdkosh.com/dictionary/english-kashmiri/
2.  **Kashmiri Language Data (GitHub)**: Repositories for Indo-Aryan linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Kashmiri Section)**: Community entries for the Kashmiri language.
    - URL: https://en.wiktionary.org/wiki/Category:Kashmiri_language
