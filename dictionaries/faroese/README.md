# Faroese Dictionary Sources

Potential sources for Faroese dictionary data:

1.  **Sprotin**: The most comprehensive online Faroese dictionary portal.
    - URL: https://sprotin.fo/
2.  **FreeDict (GitHub)**: Faroese dictionary projects for digital use.
    - URL: https://github.com/freedict/
3.  **Wiktionary (Faroese Section)**: Collaborative dictionary with Faroese definitions and translations.
    - URL: https://en.wiktionary.org/wiki/Category:Faroese_language
