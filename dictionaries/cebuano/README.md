# Cebuano Dictionary Sources

Potential sources for Cebuano dictionary data:

1.  **Binisaya.com**: A comprehensive Cebuano (Binisaya) dictionary and language portal.
    - URL: http://www.binisaya.com/
2.  **Cebuano Word Lists (GitHub)**: Repositories containing Philippine language data.
    - URL: https://github.com/
3.  **Wiktionary (Cebuano Section)**: Open-source entries for the Cebuano language.
    - URL: https://en.wiktionary.org/wiki/Category:Cebuano_language
