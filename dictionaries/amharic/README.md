# Amharic Dictionary Sources

Potential sources for Amharic dictionary data:

1.  **AmharicDictionary.com**: A comprehensive online Amharic dictionary.
    - URL: http://www.amharicdictionary.com/
2.  **Amharic Language Projects (GitHub)**: Repositories for Ethiopian linguistic data.
    - URL: https://github.com/
3.  **Wiktionary (Amharic Section)**: Collaborative dictionary entries for the Amharic language.
    - URL: https://en.wiktionary.org/wiki/Category:Amharic_language
