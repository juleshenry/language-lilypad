# Punjabi Dictionary Sources

Potential sources for Punjabi dictionary data:

1.  **Sri Granth**: Punjabi (Gurmukhi) dictionary and linguistic resource.
    - URL: https://www.srigranth.org/
2.  **English-Punjabi Dictionary (GitHub)**: Repositories for Punjabi word lists.
    - URL: https://github.com/
3.  **Wiktionary (Punjabi Section)**: Collaborative dictionary entries for the Punjabi language.
    - URL: https://pa.wiktionary.org/
