# Lao Dictionary Sources

Potential sources for Lao dictionary data:

1.  **SEAlang Lao Dictionary**: Comprehensive online Lao-English dictionary.
    - URL: http://sealang.net/lao/dictionary.htm
2.  **Lao Language Data (GitHub)**: Repositories for Southeast Asian linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Lao Section)**: Collaborative entries for the Lao language.
    - URL: https://lo.wiktionary.org/
