# Marathi Dictionary Sources

Potential sources for Marathi dictionary data:

1.  **Marathi-English Dictionary (Morya)**: Popular online Marathi dictionary portal.
    - URL: http://morya.com/marathi-dictionary/
2.  **FreeDict (GitHub)**: Marathi-English dictionary data.
    - URL: https://github.com/freedict/fd-mar-eng
3.  **Wiktionary (Marathi Section)**: Community entries for the Marathi language.
    - URL: https://mr.wiktionary.org/
