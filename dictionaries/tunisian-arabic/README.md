# Tunisian Arabic Dictionary Sources

Potential sources for Tunisian Arabic (Derja) dictionary data:

1.  **Tunisian Arabic Dictionary (Glosbe)**: Multilingual online dictionary portal.
    - URL: https://glosbe.com/aeb/en
2.  **Tunisian Arabic Projects (GitHub)**: Repositories for Maghrebi Arabic dialects.
    - URL: https://github.com/
3.  **Wiktionary (Tunisian Arabic Section)**: Community entries for the Tunisian Arabic dialect.
    - URL: https://en.wiktionary.org/wiki/Category:Tunisian_Arabic_language
