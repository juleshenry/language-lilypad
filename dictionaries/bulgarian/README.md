# Bulgarian Dictionary Sources

Potential sources for Bulgarian dictionary data:

1.  **Rechnik.info**: A popular online Bulgarian dictionary and language portal.
    - URL: https://rechnik.info/
2.  **FreeDict (GitHub)**: Bulgarian-English dictionary pairs.
    - URL: https://github.com/freedict/fd-bul-eng
3.  **Wiktionary (Bulgarian Section)**: Collaborative dictionary with extensive Bulgarian entries.
    - URL: https://bg.wiktionary.org/
