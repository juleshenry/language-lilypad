# Khmer Dictionary Sources

Potential sources for Khmer dictionary data:

1.  **Chuon Nath Khmer Dictionary**: The authoritative Khmer dictionary portal.
    - URL: http://dictionary.tnaot.com/
2.  **Khmer Language Data (GitHub)**: Repositories for Southeast Asian linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Khmer Section)**: Collaborative entries for the Khmer language.
    - URL: https://km.wiktionary.org/
