# Georgian Dictionary Sources

Potential sources for Georgian dictionary data:

1.  **GPC (Georgian Public Center)**: The official Georgian dictionary portal.
    - URL: http://www.ena.ge/
2.  **FreeDict (GitHub)**: Georgian-English dictionary data.
    - URL: https://github.com/freedict/fd-kat-eng
3.  **Wiktionary (Georgian Section)**: Open-source entries for the Georgian language.
    - URL: https://ka.wiktionary.org/
