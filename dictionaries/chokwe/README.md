# Chokwe Dictionary Sources

Potential sources for Chokwe dictionary data:

1.  **Chokwe Dictionary (Glosbe)**: Large multilingual dictionary portal.
    - URL: https://glosbe.com/cjk/en
2.  **African Language Projects (GitHub)**: Repositories containing Chokwe word lists.
    - URL: https://github.com/
3.  **Wiktionary (Chokwe Section)**: Collaborative entries for the Chokwe language.
    - URL: https://en.wiktionary.org/wiki/Category:Chokwe_language
