# Croatian Dictionary Sources

Potential sources for Croatian dictionary data:

1.  **Hrvatski jezični portal (HJP)**: The most comprehensive online Croatian dictionary portal.
    - URL: http://hjp.znanje.hr/
2.  **FreeDict (GitHub)**: Croatian-English and other language pairs.
    - URL: https://github.com/freedict/fd-hrv-eng
3.  **Wiktionary (Croatian Section)**: A collaborative open-source Croatian dictionary.
    - URL: https://en.wiktionary.org/wiki/Category:Croatian_language
