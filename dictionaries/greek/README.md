# Greek Dictionary Sources

Potential sources for Greek dictionary data:

1.  **Triantafyllidis Dictionary**: Standard Modern Greek dictionary portal.
    - URL: http://www.greek-language.gr/greekLang/modern_greek/tools/lexica/triantafyllides/
2.  **FreeDict (GitHub)**: Greek-English dictionary pairs.
    - URL: https://github.com/freedict/fd-ell-eng
3.  **Wiktionary (Greek Section)**: Extensive coverage of Modern and Ancient Greek.
    - URL: https://en.wiktionary.org/wiki/Category:Greek_language
