# Czech Dictionary Sources

Potential sources for Czech dictionary data:

1.  **Slovník spisovné češtiny (SSČ)**: Official modern Czech dictionary portal.
    - URL: https://slovnikcestiny.cz/
2.  **FreeDict (GitHub)**: Czech-English dictionary pairs.
    - URL: https://github.com/freedict/fd-ces-eng
3.  **Wiktionary (Czech Section)**: Comprehensive open-source Czech dictionary.
    - URL: https://cs.wiktionary.org/
