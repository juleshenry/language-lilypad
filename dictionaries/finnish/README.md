# Finnish Dictionary Sources

Potential sources for Finnish dictionary data:

1.  **Kielitoimiston sanakirja**: The official dictionary of modern Finnish.
    - URL: https://www.kielitoimistonsanakirja.fi/
2.  **Sanakirja.org**: A multilingual Finnish dictionary portal.
    - URL: https://www.sanakirja.org/
3.  **Wiktionary (Finnish Section)**: Open-source Finnish dictionary with extensive entries.
    - URL: https://en.wiktionary.org/wiki/Category:Finnish_language
