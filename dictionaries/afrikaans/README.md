# Afrikaans Dictionary Sources

Potential sources for Afrikaans dictionary data:

1.  **ViviA (Virtuele Instituut vir Afrikaans)**: Comprehensive online Afrikaans dictionary portal.
    - URL: https://viva-afrikaans.org/
2.  **FreeDict (GitHub)**: Afrikaans-English dictionary pairs.
    - URL: https://github.com/freedict/fd-afr-eng
3.  **Wiktionary (Afrikaans Section)**: Large collection of Afrikaans words and definitions.
    - URL: https://af.wiktionary.org/
