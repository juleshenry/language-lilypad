# Esperanto Dictionary Sources

Potential sources for Esperanto dictionary data:

1.  **Vortaro.net (Plena Ilustrita Vortaro)**: The most authoritative Esperanto dictionary.
    - URL: https://vortaro.net/
2.  **FreeDict (GitHub)**: Esperanto-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-epo-eng
3.  **Wiktionary (Esperanto Section)**: A large collection of Esperanto words and translations.
    - URL: https://en.wiktionary.org/wiki/Category:Esperanto_language
