# Kannada Dictionary Sources

Potential sources for Kannada dictionary data:

1.  **Alar Kannada Dictionary**: A comprehensive online Kannada dictionary portal.
    - URL: https://alar.ijm.nu/
2.  **FreeDict (GitHub)**: Kannada-English dictionary data.
    - URL: https://github.com/freedict/fd-kan-eng
3.  **Wiktionary (Kannada Section)**: Extensive open-source Kannada dictionary entries.
    - URL: https://kn.wiktionary.org/
