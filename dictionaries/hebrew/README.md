# Hebrew Dictionary Sources

Potential sources for Hebrew dictionary data:

1.  **Morfix**: A widely used Hebrew-English / English-Hebrew online dictionary.
    - URL: https://www.morfix.co.il/
2.  **Open Source Hebrew Dictionary (GitHub)**: Various open-source projects for Hebrew linguistic data.
    - URL: https://github.com/project-hebrew/dictionary
3.  **Wiktionary (Hebrew Section)**: Collaborative dictionary with modern and ancient Hebrew entries.
    - URL: https://en.wiktionary.org/wiki/Category:Hebrew_language
