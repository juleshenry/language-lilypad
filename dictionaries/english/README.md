# English Dictionary Sources

Potential sources for English dictionary data:

1.  **Princeton WordNet**: A large lexical database of English.
    - URL: https://wordnet.princeton.edu/
2.  **FreeDict (GitHub)**: Various English dictionary datasets and word lists.
    - URL: https://github.com/freedict/
3.  **Wiktionary (English Section)**: The primary collaborative English dictionary project.
    - URL: https://en.wiktionary.org/
