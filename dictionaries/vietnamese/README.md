# Vietnamese Dictionary Sources

Potential sources for Vietnamese dictionary data:

1.  **VNDic.net**: A popular online Vietnamese dictionary portal.
    - URL: http://vndic.net/
2.  **FreeDict (GitHub)**: Vietnamese-English dictionary pairs.
    - URL: https://github.com/freedict/fd-vie-eng
3.  **Wiktionary (Vietnamese Section)**: Large collection of Vietnamese lexical data.
    - URL: https://vi.wiktionary.org/
