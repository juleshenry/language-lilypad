# Tosk Albanian Dictionary Sources

Potential sources for Tosk Albanian dictionary data:

1.  **Fjalor.shkenca.org**: Digital dictionary portal for the Albanian language.
    - URL: http://fjalor.shkenca.org/
2.  **Albanian Language Data (GitHub)**: Repositories for Balkan linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Albanian Section)**: Collaborative entries for the Albanian language (Tosk being the standard).
    - URL: https://en.wiktionary.org/wiki/Category:Albanian_language
