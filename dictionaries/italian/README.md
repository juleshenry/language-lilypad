# Italian Dictionary Sources

Potential sources for Italian dictionary data:

1.  **Treccani**: The most authoritative Italian linguistic and encyclopedic portal.
    - URL: https://www.treccani.it/vocabolario/
2.  **FreeDict (GitHub)**: Italian-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-ita-eng
3.  **Wiktionary (Italian Section)**: Large collection of Italian lexical data.
    - URL: https://it.wiktionary.org/
