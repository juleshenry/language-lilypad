# Central Aymara Dictionary Sources

Potential sources for Aymara dictionary data:

1.  **Aymara-Spanish Dictionary (Glosbe)**: Multilingual online dictionary for Andean languages.
    - URL: https://glosbe.com/aym/es
2.  **Aymara Language Projects (GitHub)**: Repositories for South American indigenous language data.
    - URL: https://github.com/
3.  **Wiktionary (Aymara Section)**: Collaborative entries for the Aymara language.
    - URL: https://en.wiktionary.org/wiki/Category:Aymara_language
