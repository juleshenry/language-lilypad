# Mesopotamian Arabic Dictionary Sources

Potential sources for Mesopotamian Arabic (Iraqi) dictionary data:

1.  **Iraqi Arabic Dictionary (Glosbe)**: Multilingual online dictionary for Mesopotamian Arabic.
    - URL: https://glosbe.com/acm/en
2.  **Arabic Dialect Projects (GitHub)**: Repositories for Iraqi Arabic linguistic data.
    - URL: https://github.com/
3.  **Wiktionary (Mesopotamian Arabic Section)**: Collaborative entries for the Mesopotamian Arabic dialect.
    - URL: https://en.wiktionary.org/wiki/Category:Mesopotamian_Arabic_language
