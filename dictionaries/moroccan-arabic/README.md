# Moroccan Arabic Dictionary Sources

Potential sources for Moroccan Arabic (Darija) dictionary data:

1.  **Moroccan Arabic Dictionary (Glosbe)**: Large multilingual dictionary portal.
    - URL: https://glosbe.com/ary/en
2.  **Moroccan Arabic Projects (GitHub)**: Repositories for Maghrebi Arabic dialects.
    - URL: https://github.com/
3.  **Wiktionary (Moroccan Arabic Section)**: Community entries for the Moroccan Arabic language.
    - URL: https://en.wiktionary.org/wiki/Category:Moroccan_Arabic_language
