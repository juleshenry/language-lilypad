# Bosnian Dictionary Sources

Potential sources for Bosnian dictionary data:

1.  **Rječnik.ba**: A popular online Bosnian-English dictionary.
    - URL: https://www.rjecnik.ba/
2.  **FreeDict (GitHub)**: Bosnian dictionary projects for open-source use.
    - URL: https://github.com/freedict/
3.  **Wiktionary (Bosnian Section)**: Collaborative dictionary for the Bosnian language.
    - URL: https://bs.wiktionary.org/
