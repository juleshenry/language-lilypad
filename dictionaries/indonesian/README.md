# Indonesian Dictionary Sources

Potential sources for Indonesian (Bahasa Indonesia) dictionary data:

1.  **KBBI (Kamus Besar Bahasa Indonesia)**: The official dictionary of the Indonesian language. Often accessible via web interfaces or unofficial APIs.
    - URL: https://kbbi.kemdikbud.go.id/
2.  **FreeDict (GitHub)**: A collection of free dictionaries, including Indonesian-English and others.
    - URL: https://github.com/freedict/fd-ind-eng
3.  **Wiktionary (Indonesian Section)**: A collaborative project to produce a free-content multilingual dictionary.
    - URL: https://en.wiktionary.org/wiki/Category:Indonesian_language
