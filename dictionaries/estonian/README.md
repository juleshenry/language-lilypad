# Estonian Dictionary Sources

Potential sources for Estonian dictionary data:

1.  **Sõnaveeb (EKI)**: The central portal of the Institute of the Estonian Language.
    - URL: https://sonaveeb.ee/
2.  **FreeDict (GitHub)**: Estonian dictionary data for software developers.
    - URL: https://github.com/freedict/fd-est-eng
3.  **Wiktionary (Estonian Section)**: Comprehensive collaborative Estonian dictionary.
    - URL: https://en.wiktionary.org/wiki/Category:Estonian_language
