# Bambara Dictionary Sources

Potential sources for Bambara (Bamanankan) dictionary data:

1.  **Bambara-French Dictionary (Bambara.org)**: A key online resource for the Bambara language.
    - URL: http://www.bambara.org/
2.  **Bambara Language Resources (GitHub)**: Repositories for Mande language data.
    - URL: https://github.com/
3.  **Wiktionary (Bambara Section)**: Community-contributed entries for the Bambara language.
    - URL: https://en.wiktionary.org/wiki/Category:Bambara_language
