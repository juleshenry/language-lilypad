# Dutch Dictionary Sources

Potential sources for Dutch dictionary data:

1.  **Van Dale / Woorden.org**: Reputable Dutch dictionary portals.
    - URL: https://www.woorden.org/
2.  **FreeDict (GitHub)**: Dutch-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-nld-eng
3.  **Wiktionary (Dutch Section)**: Large collection of Dutch lexical data.
    - URL: https://nl.wiktionary.org/
