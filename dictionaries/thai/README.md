# Thai Dictionary Sources

Potential sources for Thai dictionary data:

1.  **Thai-Language.com**: A comprehensive online Thai-English dictionary.
    - URL: http://www.thai-language.com/
2.  **Longdo Dictionary**: Popular Thai online dictionary portal.
    - URL: https://dict.longdo.com/
3.  **Wiktionary (Thai Section)**: Large collaborative dictionary for the Thai language.
    - URL: https://th.wiktionary.org/
