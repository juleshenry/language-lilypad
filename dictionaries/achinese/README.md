# Achinese Dictionary Sources

Potential sources for Achinese (Acehnese) dictionary data:

1.  **Achinese-Indonesian Dictionary (Glosbe)**: Large multilingual dictionary portal.
    - URL: https://glosbe.com/ace/id
2.  **Achinese Language Resources (GitHub)**: Repositories for Sumatran linguistic data.
    - URL: https://github.com/
3.  **Wiktionary (Achinese Section)**: Community entries for the Achinese language.
    - URL: https://en.wiktionary.org/wiki/Category:Achinese_language
