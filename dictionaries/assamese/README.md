# Assamese Dictionary Sources

Potential sources for Assamese dictionary data:

1.  **Xobdo.org**: A multilingual dictionary portal focusing on languages of North East India.
    - URL: http://xobdo.org/
2.  **Assamese-English Dictionary (GitHub)**: Repositories for Assamese linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Assamese Section)**: Community entries for the Assamese language.
    - URL: https://as.wiktionary.org/
