# Welsh Dictionary Sources

Potential sources for Welsh dictionary data:

1.  **Geiriadur Prifysgol Cymru (GPC)**: The standard historical Welsh dictionary.
    - URL: http://welsh-dictionary.ac.uk/gpc/gpc.html
2.  **Eurfa (GitHub)**: Open-source Welsh dictionary and language resources.
    - URL: https://github.com/
3.  **Wiktionary (Welsh Section)**: Collaborative dictionary for the Welsh language.
    - URL: https://en.wiktionary.org/wiki/Category:Welsh_language
