# Egyptian Arabic Dictionary Sources

Potential sources for Egyptian Arabic dictionary data:

1.  **Lisaan Masry**: A comprehensive online Egyptian Arabic dictionary.
    - URL: https://lisaanmasry.org/
2.  **Egyptian Arabic Resources (GitHub)**: Repositories for dialectal Arabic datasets.
    - URL: https://github.com/
3.  **Wiktionary (Egyptian Arabic Section)**: Collaborative entries for the Egyptian Arabic dialect.
    - URL: https://en.wiktionary.org/wiki/Category:Egyptian_Arabic_language
