# Turkish Dictionary Sources

Potential sources for Turkish dictionary data:

1.  **TDK Sözlük**: The official dictionary of the Turkish Language Association.
    - URL: https://sozluk.gov.tr/
2.  **FreeDict (GitHub)**: Turkish-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-tur-eng
3.  **Wiktionary (Turkish Section)**: Large collection of Turkish lexical entries.
    - URL: https://tr.wiktionary.org/
