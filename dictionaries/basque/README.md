# Basque Dictionary Sources

Potential sources for Basque dictionary data:

1.  **Elhuyar Hiztegia**: A highly reputable Basque-Spanish / Basque-English dictionary.
    - URL: https://hiztegiak.elhuyar.eus/
2.  **FreeDict (GitHub)**: Basque-English dictionary pairs.
    - URL: https://github.com/freedict/fd-eus-eng
3.  **Wiktionary (Basque Section)**: Collaborative dictionary for the Basque language (Euskara).
    - URL: https://en.wiktionary.org/wiki/Category:Basque_language
