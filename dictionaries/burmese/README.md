# Burmese Dictionary Sources

Potential sources for Burmese dictionary data:

1.  **Myanmar-English Dictionary (SEAlang)**: Comprehensive online Burmese dictionary.
    - URL: http://sealang.net/burmese/dictionary.htm
2.  **Burmese Language Data (GitHub)**: Repositories for Southeast Asian linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Burmese Section)**: Collaborative entries for the Burmese language.
    - URL: https://en.wiktionary.org/wiki/Category:Burmese_language
