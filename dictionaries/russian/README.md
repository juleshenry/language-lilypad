# Russian Dictionary Sources

Potential sources for Russian dictionary data:

1.  **Gramota.ru**: The authoritative Russian linguistic portal.
    - URL: http://gramota.ru/
2.  **FreeDict (GitHub)**: Large collection of Russian-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-rus-eng
3.  **Wiktionary (Russian Section)**: One of the largest and most active Wiktionary sections.
    - URL: https://ru.wiktionary.org/
