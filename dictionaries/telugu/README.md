# Telugu Dictionary Sources

Potential sources for Telugu dictionary data:

1.  **Telugu-English Dictionary (Andhrabharati)**: Comprehensive online Telugu dictionary.
    - URL: http://www.andhrabharati.com/dictionary/
2.  **FreeDict (GitHub)**: Telugu-English dictionary data.
    - URL: https://github.com/freedict/fd-tel-eng
3.  **Wiktionary (Telugu Section)**: Open-source entries for the Telugu language.
    - URL: https://te.wiktionary.org/
