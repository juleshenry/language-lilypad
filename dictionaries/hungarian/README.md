# Hungarian Dictionary Sources

Potential sources for Hungarian dictionary data:

1.  **Sztaki Szótár**: A popular and long-running Hungarian-English online dictionary.
    - URL: https://szotar.sztaki.hu/
2.  **FreeDict (GitHub)**: Hungarian-English dictionary files in TEI format.
    - URL: https://github.com/freedict/fd-hun-eng
3.  **Wiktionary (Hungarian Section)**: Extensive collection of Hungarian words and definitions.
    - URL: https://en.wiktionary.org/wiki/Category:Hungarian_language
