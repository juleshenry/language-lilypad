# Urdu Dictionary Sources

Potential sources for Urdu dictionary data:

1.  **Urdu Lughat**: The official Urdu dictionary portal.
    - URL: http://www.urdulughat.info/
2.  **Rekhta Dictionary**: A popular dictionary for Urdu poetry and literature.
    - URL: https://www.rekhta.org/dictionary
3.  **Wiktionary (Urdu Section)**: Collaborative entries for the Urdu language.
    - URL: https://en.wiktionary.org/wiki/Category:Urdu_language
