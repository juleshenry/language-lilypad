# Belarusian Dictionary Sources

Potential sources for Belarusian dictionary data:

1.  **Slovnik.org**: A comprehensive Belarusian dictionary portal.
    - URL: https://www.slovnik.org/
2.  **FreeDict (GitHub)**: Belarusian-English dictionary data.
    - URL: https://github.com/freedict/fd-bel-eng
3.  **Wiktionary (Belarusian Section)**: Collaborative dictionary for the Belarusian language.
    - URL: https://be.wiktionary.org/
