# Fijian Dictionary Sources

Potential sources for Fijian dictionary data:

1.  **Na vosa vakaviti (Wiktionary)**: Open-source entries for the Fijian language.
    - URL: https://en.wiktionary.org/wiki/Category:Fijian_language
2.  **University of the South Pacific (USP)**: Academic linguistic resources for Fijian.
    - URL: https://www.usp.ac.fj/
3.  **Glosbe**: Multilingual online dictionary supporting Fijian-English.
    - URL: https://glosbe.com/fj/en
