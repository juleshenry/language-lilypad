# Ewe Dictionary Sources

Potential sources for Ewe dictionary data:

1.  **Ewe-English Dictionary (GitHub)**: Various word lists and dictionary datasets.
    - URL: https://github.com/
2.  **Wiktionary (Ewe Section)**: Open-source entries for the Ewe language.
    - URL: https://en.wiktionary.org/wiki/Category:Ewe_language
3.  **Aglan-Dictionary**: A collection of Ewe language resources.
    - URL: https://glosbe.com/ee/en
