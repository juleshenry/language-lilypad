# Central Kurdish (Sorani) Dictionary Sources

Potential sources for Sorani Kurdish dictionary data:

1.  **VejinLex**: A digital dictionary for Kurdish dialects.
    - URL: https://lex.vejin.net/en
2.  **Sorani-English Dictionary (GitHub)**: Repositories for Kurdish language datasets.
    - URL: https://github.com/
3.  **Wiktionary (Sorani Kurdish Section)**: Open-source entries for Central Kurdish.
    - URL: https://en.wiktionary.org/wiki/Category:Central_Kurdish_language
