# Akan Dictionary Sources

Potential sources for Akan (Twi/Fante) dictionary data:

1.  **Akan-English Dictionary (Glosbe)**: Multilingual online dictionary for West African languages.
    - URL: https://glosbe.com/ak/en
2.  **Akan Language Projects (GitHub)**: Repositories for Ghanaian linguistic data.
    - URL: https://github.com/
3.  **Wiktionary (Akan Section)**: Community entries for the Akan language.
    - URL: https://en.wiktionary.org/wiki/Category:Akan_language
