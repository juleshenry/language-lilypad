# Ukrainian Dictionary Sources

Potential sources for Ukrainian dictionary data:

1.  **Slovnyk.ua**: A comprehensive online Ukrainian dictionary portal.
    - URL: https://slovnyk.ua/
2.  **FreeDict (GitHub)**: Ukrainian-English dictionary data.
    - URL: https://github.com/freedict/fd-ukr-eng
3.  **Wiktionary (Ukrainian Section)**: Extensive collaborative Ukrainian dictionary.
    - URL: https://uk.wiktionary.org/
