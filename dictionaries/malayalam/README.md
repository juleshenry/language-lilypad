# Malayalam Dictionary Sources

Potential sources for Malayalam dictionary data:

1.  **Malayalam Lexicon (University of Kerala)**: The authoritative Malayalam dictionary portal.
    - URL: http://www.malayalamlexicon.kerala.gov.in/
2.  **FreeDict (GitHub)**: Malayalam-English dictionary data.
    - URL: https://github.com/freedict/fd-mal-eng
3.  **Wiktionary (Malayalam Section)**: Extensive open-source Malayalam dictionary entries.
    - URL: https://ml.wiktionary.org/
