# Bashkir Dictionary Sources

Potential sources for Bashkir dictionary data:

1.  **Bashkir-Russian Dictionary (Bashkort.com)**: Online portal for the Bashkir language.
    - URL: http://bashkort.com/
2.  **Bashkir Language Data (GitHub)**: Repositories for Turkic linguistic resources.
    - URL: https://github.com/
3.  **Wiktionary (Bashkir Section)**: Collaborative dictionary entries for the Bashkir language.
    - URL: https://en.wiktionary.org/wiki/Category:Bashkir_language
