# Korean Dictionary Sources

Potential sources for Korean dictionary data:

1.  **Naver Dictionary**: The most widely used online Korean dictionary portal.
    - URL: https://dict.naver.com/
2.  **National Institute of Korean Language (GitHub)**: Digital resources for the Korean language.
    - URL: https://github.com/
3.  **Wiktionary (Korean Section)**: Large collection of Korean lexical entries and translations.
    - URL: https://ko.wiktionary.org/
