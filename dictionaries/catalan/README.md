# Catalan Dictionary Sources

Potential sources for Catalan dictionary data:

1.  **Diccionari de la llengua catalana (DIEC2)**: The official dictionary of the Institut d'Estudis Catalans.
    - URL: https://dlc.iec.cat/
2.  **FreeDict (GitHub)**: Catalan-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-cat-eng
3.  **Wiktionary (Catalan Section)**: Large collection of Catalan lexical data.
    - URL: https://ca.wiktionary.org/
