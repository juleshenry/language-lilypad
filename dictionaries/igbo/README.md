# Igbo Dictionary Sources

Potential sources for Igbo dictionary data:

1.  **Nkọwa okwu (Igbo Dictionary)**: A comprehensive online Igbo-English dictionary.
    - URL: https://nkowaokwu.com/
2.  **IgboNet (GitHub)**: A repository containing Igbo language resources and dictionaries.
    - URL: https://github.com/igbonet/igbo-dictionary
3.  **Columbia University Igbo Language Resources**: Academic resources and word lists for the Igbo language.
    - URL: http://www.columbia.edu/itc/ealac/lan/igbo/
