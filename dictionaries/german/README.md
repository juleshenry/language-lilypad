# German Dictionary Sources

Potential sources for German dictionary data:

1.  **Duden / DWDS**: Authoritative German linguistic portals.
    - URL: https://www.duden.de/
2.  **FreeDict (GitHub)**: Extensive German-English and other bilingual dictionaries.
    - URL: https://github.com/freedict/fd-deu-eng
3.  **Wiktionary (German Section)**: Huge collection of German lexical data.
    - URL: https://de.wiktionary.org/
